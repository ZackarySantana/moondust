# Multi-device and shared server

This is a **planned** direction, not something Moondust ships today.

The idea is to let one Moondust workspace or project state sync across machines, or to connect multiple clients to a shared backend so teams can coordinate agent work without everyone on a single desktop session.

Details (protocol, hosting, and security model) are still TBA. When we have a concrete design, this page will be updated.
