# Welcome

Moondust is a desktop app for AI coding agents: multiple **projects** (workspaces), **threads** per project, and a layout built for real repositories with **chat** (several upstream options), a full **Git and review sidebar**, diffs, helper lanes, and a **terminal** in one place, not a single floating chat window.

**Chat providers:** connect **OpenRouter** (OAuth or API key), or use **Cursor** and **Claude** via their official CLIs on your machine. The [Providers](providers/overview) page summarizes status and links to setup for each.

**Git:** stage, diff, branch review, a **commit graph** against your default branch, and a **Git wizard** for rebase/merge, including optional AI-assisted conflict resolution. See [Git and review sidebar](features/git-sidebar).

## Roadmap

Longer-term ideas are documented as **plans** in this site; for example [Multi-device and shared server](plans/multi-device-shared-server) (not shipped; design TBA).

Some providers (e.g. Codex, Pi) remain **planned**; see the [provider overview](providers/overview) table.

Feedback and contributions are welcome via **GitHub Issues and Discussions** on the repository. For building from source and doc development, see [Contributing](contributing).
