# Welcome

Moondust is a desktop app for AI coding agents: multiple **projects** (workspaces), **threads** per project, and a layout built for real repositories—chat, git review, diffs, and a terminal in one place—not a single floating chat window.

Configure **OpenRouter** (OAuth or API key) under Settings to enable chat; see [Providers](providers/overview) for status of other planned providers.

## Roadmap

Longer-term ideas are documented as **plans** in this site—for example [Multi-device and shared server](plans/multi-device-shared-server) (not shipped; design TBA).

Additional model providers beyond OpenRouter are **planned**; the [provider overview](providers/overview) table tracks what is implemented vs placeholder.

Feedback and contributions are welcome via **GitHub Issues and Discussions** on the repository. For building from source and doc development, see [Contributing](contributing).
