# Storybook

We publish a **[Storybook](https://storybook.js.org/)** build of selected UI components from the desktop app, useful for reviewing components in isolation without running Wails.

**Open the catalog:** [docs.moondust.pro/storybook](https://docs.moondust.pro/storybook)

The static build lives under [`frontend/.storybook/`](https://github.com/zackarysantana/moondust/tree/main/frontend/.storybook) and `*.stories.tsx` in [`frontend/src/`](https://github.com/zackarysantana/moondust/tree/main/frontend/src); it is deployed next to this site from the same [docs workflow](https://github.com/zackarysantana/moondust/blob/main/.github/workflows/docs-deploy.yml).
