# Workspaces

Projects live in workspaces. Each workspace has one root path, agent context, tools, and file scope follow that root.

## Inside a workspace

- Threads scoped to the workspace.
- Git state associated with the workspace.
- BYOT - Bring your own terminal.

## Multiple projects

Multiple projects live in multiple workspaces, these all coexist in the same Moondust instance.
